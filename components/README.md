# Components

Built from primitives. These are recognizable UI pieces that combine multiple atoms.

## What goes here

- **cards/** — Content cards, profile cards, pricing cards, image cards, stat cards
- **modals/** — Confirmation dialogs, info modals, form modals, full-screen modals
- **dropdowns/** — Menu dropdowns, select dropdowns, multi-select
- **tabs/** — Horizontal tabs, vertical tabs, pill tabs, underline tabs
- **accordions/** — Collapsible sections, FAQ accordions
- **alerts/** — Success, error, warning, info banners and toasts
- **tables/** — Data tables, sortable tables, striped tables
- **lists/** — Ordered, unordered, description lists, action lists
- **menus/** — Context menus, action menus, kebab menus
- **pagination/** — Page numbers, prev/next, infinite scroll triggers
- **breadcrumbs/** — Navigation breadcrumb trails
- **steppers/** — Multi-step progress indicators
- **carousels/** — Image sliders, testimonial carousels
- **popovers/** — Rich content popovers, info popovers
