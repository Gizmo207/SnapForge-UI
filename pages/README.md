# Pages

Complete page templates ready to drop into a project. Each one combines a layout + patterns + components.

## What goes here

- **landing/** — Marketing landing pages
- **login/** — Login / sign-in pages
- **signup/** — Registration pages
- **dashboard/** — Admin/user dashboard pages
- **settings/** — Settings / preferences pages
- **profile/** — User profile pages
- **pricing/** — Pricing comparison pages
- **404/** — Not found error pages
- **500/** — Server error pages
- **coming-soon/** — Placeholder / under construction pages
- **blog/** — Blog listing and blog post pages
- **about/** — About us pages
- **contact/** — Contact pages
