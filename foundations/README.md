# Foundations

Design tokens and raw design decisions. Everything else in the library references these.

## What goes here

- **colors/** — Color palettes, theme colors, semantic colors (success, error, warning)
- **typography/** — Font families, font sizes, line heights, font weights
- **spacing/** — Spacing scales, margin/padding tokens
- **shadows/** — Box shadow definitions
- **borders/** — Border radius, border widths
- **breakpoints/** — Responsive breakpoint definitions
- **animations/** — Keyframes, easing functions, transition durations
