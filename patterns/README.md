# Patterns

Recognizable UI sections composed from components. These are the chunks that make up a page.

## What goes here

- **navbars/** — Top navigation bars, sticky navs, transparent navs
- **headers/** — Page headers, section headers, hero headers
- **footers/** — Simple footers, multi-column footers, sticky footers
- **heroes/** — Hero sections with CTA, image heroes, video heroes
- **forms/** — Login forms, signup forms, contact forms, search forms, multi-step forms
- **sidebars/** — Navigation sidebars, filter sidebars, collapsible sidebars
- **sections/** — Feature sections, testimonial sections, pricing sections, CTA sections
- **search/** — Search bars with results, command palettes, filter bars
- **auth/** — Login blocks, signup blocks, forgot password blocks
- **empty-states/** — No data, no results, error states, onboarding prompts
- **notifications/** — Notification panels, toast stacks, alert banners
