# Layouts

Page-level shells. These define where things go on screen — the skeleton you pour content into.

## What goes here

- **blank/** — Empty page with just a container. The starting point.
- **centered/** — Content centered on screen (great for login, error pages)
- **sidebar/** — Fixed sidebar + main content area
- **dashboard/** — Sidebar + top nav + main content grid
- **split/** — Two-column split (e.g., image left, form right)
- **stacked/** — Full-width sections stacked vertically (landing pages)
- **holy-grail/** — Header + sidebar + main + sidebar + footer
- **responsive-grid/** — CSS grid templates that adapt to screen size
