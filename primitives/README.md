# Primitives

The smallest building blocks. Each one does one thing. Many variants per type.

## What goes here

- **buttons/** — Primary, secondary, ghost, outline, icon, loading, disabled
- **inputs/** — Text, password, search, textarea, number, date
- **badges/** — Status badges, notification dots, tags, chips
- **avatars/** — Image avatars, initials, avatar groups
- **icons/** — Custom icon components or icon wrappers
- **links/** — Styled anchor tags, nav links, breadcrumb links
- **dividers/** — Horizontal rules, vertical separators
- **loaders/** — Spinners, skeleton screens, progress bars
- **toggles/** — Switches, checkboxes, radio buttons
- **labels/** — Form labels, floating labels, helper text
- **tooltips/** — Simple tooltip wrappers
- **images/** — Responsive image wrappers, lazy-load images
