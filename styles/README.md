# Styles

Shared stylesheets, CSS variables, Tailwind config, and global resets used across the library.

## What goes here

- **reset.css** — CSS reset / normalize
- **globals.css** — Global styles, CSS custom properties for tokens
- **tailwind.config.js** — Shared Tailwind configuration (if using Tailwind)
- **themes/** — Light/dark theme definitions, custom theme files
- **utilities.css** — Reusable utility classes not covered by Tailwind
